#使用的是webpack4+react—+react-router4+react-redux+antd+axios 一系列工具

##目录结构
- src
	- axios （ajax封装）
	- component （各种组件 比如面包屑 头部尾部 导航栏）
	- pages （内部页面）
	- redux （具体页面）
	  - aciton
	  - reducer
	  - store （redux的store和aciton）
	- routes （内部页面的路由处理结构）
	- style （默认和全局样式）
	-authrouter （权限认证路由页面）
